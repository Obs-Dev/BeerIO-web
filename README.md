# Beerio Web Version #

This is the web app version of the beerIO application.

### What is this repository for? ###

* Anyone that will be working on the BeerIO project

### How do I get set up? ###

- Clone the repo

- Run docker: 

```
#!php
docker-compose up -d
```
- Visit http://localhost:8080/
